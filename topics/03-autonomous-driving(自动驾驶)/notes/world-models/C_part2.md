# 端到端AD集成型 论文分析报告（Part 2）

---

## **[4] GameFormer**

**机构/会议：** 南洋理工大学（NTU），NeurIPS 2023

**核心贡献：**
1. 提出基于层级博弈论（level-k game theory）的交互预测与规划统一框架，首次将认知科学的层级推理引入自动驾驶轨迹预测。
2. 设计层级 Transformer 解码器（level-k decoder），每一层利用上一层预测结果迭代精化，显式建模多智能体间的双向交互。
3. 提出交互损失（interaction loss），通过排斥势场约束智能体轨迹避免碰撞，兼顾模仿学习与交互安全。

**方法要点：**
- 编码器用 LSTM + MLP + Transformer 分别编码智能体历史状态和矢量化地图，生成共享场景上下文。
- 解码阶段设置 K 层解码器：level-0 仅基于模态嵌入独立预测；level-k 以 level-(k-1) 的预测轨迹为附加输入，经自注意力和交叉注意力更新后输出高斯混合模型（GMM）轨迹。
- 训练目标为负对数似然模仿损失与交互损失的加权组合。

**实验结果：**
- 在 Waymo 交互预测基准（WOMD）达到 SOTA 预测精度；
- 在 nuPlan 规划基准的开环与闭环测试中均优于各对比基线方法。

## **[5] Think2Drive**

**机构/会议：** 上海交通大学（SJTU），ECCV 2024

**核心贡献：**
1. 首个将模型基强化学习（model-based RL）成功应用于准真实自动驾驶基准（CARLA Leaderboard v2，含 39 个角落场景）的工作，此前无任何方法在该基准上达到 100% 路线完成率。
2. 提出 CornerCaseRepo 基准，支持按场景类别评测驾驶模型，并设计融合路线完成率、违规次数、场景密度的平衡评测指标。
3. 针对 AD 中模型基 RL 的三大挑战（策略退化、长尾分布、车头稳定性）提出系统性解决方案。

**方法要点：**
- 基于 DreamerV3 的 RSSM 架构构建紧凑隐空间世界模型，对环境转移函数建模（状态、奖励、终止预测）。
- 规划器在隐空间内"思考"驾驶而非直接交互 CARLA 物理仿真器，显著提升训练效率（单卡 A6000，3 天收敛）。
- 引入随机重初始化规划器权重防止策略退化；设计自动化场景生成器与终止优先经验回放应对长尾问题；增加转向代价函数稳定车头方向。

**实验结果：**
- CARLA v2 全部 39 个场景实现 100% 路线完成率，为文献中首次报告成功；
- 单块 A6000 GPU 训练 3 天即达专家级水平，训练效率远超 model-free 方法。

## **[6] DriveWorld**

**机构/会议：** 北京航空航天大学 / 北京大学等多机构合作，CVPR 2024

**核心贡献：**
1. 提出首个基于世界模型的视觉中心自动驾驶 4D（时空）预训练框架，弥补现有方法忽略时序特征的不足。
2. 设计记忆状态空间模型（Memory State-Space Model，MSSM），分别以动态记忆银行（Dynamic Memory Bank）建模时序动态、以静态场景传播模块（Static Scene Propagation）建模空间静态信息。
3. 引入 Task Prompt 机制，利用预训练文本编码器（BERT/CLIP）生成任务语义提示，自适应解耦不同下游任务所需特征。

**方法要点：**
- 多摄像头图像经 2D→3D 视角变换获得 BEV 特征，送入 MSSM 生成紧凑时空隐表示。
- 动态记忆银行采用运动感知层归一化（MLN），融合运动参数对隐动态进行仿射变换；静态场景传播随机采样参考帧 BEV 特征编码全局空间先验。
- 预训练目标：KL 散度（先/后验对齐）+ 交叉熵（3D 占用预测）+ L1（动作预测）联合优化。

**实验结果：**
- 在 OpenScene 数据集上预训练后，3D 目标检测 mAP +7.5%、多目标跟踪 AMOTA +3.0%、运动预测 minADE -0.1m、占用预测 IoU +3.0%、规划平均 L2 误差 -0.34m，全面优于 2D/3D 预训练基线。
