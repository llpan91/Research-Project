# 扩散模型图像生成：数据构建、训练、损失与优化

> 覆盖内容：DDPM / DDIM / Score Matching / CFG 等主流范式
> 关键参考：Ho et al. 2020 (DDPM), Song et al. 2020 (Score SDE), Ho & Salimans 2022 (CFG)

---

## 一、核心思想：两个方向的马尔可夫链

扩散模型的整个框架建立在两个方向相反的随机过程上：

```
真实图像 x₀  ──[加噪，Forward]──→  纯高斯噪声 x_T
纯高斯噪声 x_T ──[去噪，Reverse]──→  真实图像 x₀
```

- **Forward Process（扩散过程）**：固定的、无参数的，人为设计，用来构建训练样本
- **Reverse Process（去噪过程）**：有参数的神经网络来学习，是真正需要训练的部分

---

## 二、样本构建（数据预处理 / Forward Process）

### 2.1 基本公式

给定一张真实图像 $x_0$，通过 $T$ 步逐渐加入高斯噪声，得到一系列带噪图像 $x_1, x_2, \ldots, x_T$。

每一步的转移概率（条件分布）定义为：

$$q(x_t \mid x_{t-1}) = \mathcal{N}\!\left(x_t;\, \sqrt{1 - \beta_t}\, x_{t-1},\; \beta_t \mathbf{I}\right)$$

其中 $\beta_t$ 是预先设计的**噪声调度表（noise schedule）**，满足 $0 < \beta_1 < \beta_2 < \cdots < \beta_T < 1$，控制每步加入多少噪声。

### 2.2 闭式采样（训练时最重要的技巧）

不需要真的逐步执行 T 步加噪。定义：

$$\alpha_t = 1 - \beta_t, \qquad \bar{\alpha}_t = \prod_{s=1}^{t} \alpha_s \quad \text{（累积信噪比）}$$

则可以从 $x_0$ **一步直接采样任意时刻的噪声图像**：

$$q(x_t \mid x_0) = \mathcal{N}\!\left(x_t;\, \sqrt{\bar{\alpha}_t}\, x_0,\; (1 - \bar{\alpha}_t)\, \mathbf{I}\right)$$

等价的重参数化采样：

$$x_t = \sqrt{\bar{\alpha}_t}\, x_0 + \sqrt{1 - \bar{\alpha}_t}\, \varepsilon, \qquad \varepsilon \sim \mathcal{N}(0, \mathbf{I})$$

> **这是训练样本构建的核心**：给定任意真实图像 $x_0$，随机采样时间步 $t$ 和噪声 $\varepsilon$，即可立即得到对应的噪声图像 $x_t$，无需迭代。

### 2.3 噪声调度表（Noise Schedule）

噪声调度表决定了 `ᾱt` 随 `t` 的衰减曲线，直接影响训练质量。

#### Linear Schedule（Ho et al. 2020）

$\beta_t$ 在 $[\beta_{\text{start}},\, \beta_{\text{end}}]$ 之间线性插值，典型值：$\beta_{\text{start}} = 10^{-4}$，$\beta_{\text{end}} = 0.02$，$T = 1000$。

问题：在低分辨率图像上表现尚可，但高分辨率时 $x_T$ 未必足够接近纯噪声。

#### Cosine Schedule（Nichol & Dhariwal 2021，改进）

$$\bar{\alpha}_t = \cos^2\!\left(\frac{t/T + s}{1 + s} \cdot \frac{\pi}{2}\right)$$

优点：$\bar{\alpha}_t$ 在两端变化缓慢、中间快速衰减，信噪比过渡更平滑，训练更稳定。

### 2.4 图像预处理细节

| 步骤 | 操作 | 说明 |
|------|------|------|
| 像素归一化 | `[0, 255] → [-1, 1]` | 使图像均值为 0，与高斯噪声对齐 |
| 分辨率统一 | Resize / Center Crop | 保证批次内形状一致 |
| 数据增强 | 水平翻转（可选） | 轻量增强，不影响分布语义 |
| 加噪 | 按闭式公式采样 | 每次迭代在线生成，不预存 |

## 三、网络结构（学习去噪的模型）

训练的核心是一个**带时间条件的去噪网络** $\varepsilon_\theta(x_t, t)$，输入带噪图像和时间步，输出预测的噪声（或 $x_0$，见后文）。

### 3.1 主干网络：U-Net

标准选择是 **Time-conditioned U-Net**：

```
输入: x_t  [B, C, H, W]
       ↓
  DownBlock × N   (含 ResNet + Self-Attention)
       ↓
  Bottleneck      (含 Self-Attention)
       ↓
  UpBlock × N     (含 ResNet + Self-Attention + Skip Connection)
       ↓
输出: ε̂  [B, C, H, W]    （与输入同分辨率）
```

时间步 $t$ 通过 **Sinusoidal Positional Embedding → MLP → 注入每个 ResBlock** 的方式作为条件。

### 3.2 大规模生成：Latent Diffusion / DiT

- **LDM（Latent Diffusion Model, Stable Diffusion）**：先用 VAE 将图像压缩到低维隐空间，在隐空间做扩散，大幅降低计算量
- **DiT（Diffusion Transformer）**：用 Transformer 替换 U-Net，更强的全局建模能力（FLUX, SD3 采用此架构）

## 四、训练过程

### 4.1 单次训练迭代的完整流程

1. 从数据集采样真实图像：$x_0 \sim p_{\text{data}}(x)$
2. 均匀采样时间步：$t \sim \mathrm{Uniform}\{1, 2, \ldots, T\}$
3. 采样高斯噪声：$\varepsilon \sim \mathcal{N}(0, \mathbf{I})$
4. 构建噪声图像（闭式）：$x_t = \sqrt{\bar{\alpha}_t}\, x_0 + \sqrt{1 - \bar{\alpha}_t}\, \varepsilon$
5. 网络前向推断：$\hat{\varepsilon} = \varepsilon_\theta(x_t, t)$
6. 计算损失：$L = \|\varepsilon - \hat{\varepsilon}\|^2$
7. 反向传播 + 梯度下降更新 $\theta$

### 4.2 伪代码

```python
for x0 in dataloader:
    x0 = normalize(x0)                          # [-1, 1]
    t  = torch.randint(1, T+1, (B,))            # 随机时间步
    eps = torch.randn_like(x0)                  # 真实噪声

    # 闭式加噪
    sqrt_alpha_bar = sqrt_alpha_bars[t]         # 预计算好的 √ᾱt
    sqrt_one_minus = sqrt_one_minus_alpha_bars[t]
    x_t = sqrt_alpha_bar * x0 + sqrt_one_minus * eps

    # 网络预测 + 损失
    eps_pred = model(x_t, t)
    loss = F.mse_loss(eps_pred, eps)

    optimizer.zero_grad()
    loss.backward()
    optimizer.step()
```

## 五、损失函数

### 5.1 标准损失：简化的 ELBO（DDPM）

DDPM 从变分下界（ELBO）出发推导，最终将损失简化为：

$$L_{\text{simple}} = \mathbb{E}_{t,\, x_0,\, \varepsilon}\left[\left\|\varepsilon - \varepsilon_\theta(x_t, t)\right\|^2\right]$$

即：**预测噪声的 MSE 损失**。

#### 为什么能简化为 MSE？

完整的 ELBO 包含多项 KL 散度，其中关键的去噪项为：

$$L_{t-1} = \mathbb{E}_q\left[D_{\mathrm{KL}}\!\left(q(x_{t-1} \mid x_t, x_0) \;\|\; p_\theta(x_{t-1} \mid x_t)\right)\right]$$

由于 $q(x_{t-1} \mid x_t, x_0)$ 是解析可得的高斯分布，且其均值可以用 $\varepsilon$ 和 $x_t$ 表示：

$$\tilde{\mu}_t(x_t, x_0) = \frac{1}{\sqrt{\alpha_t}}\left(x_t - \frac{\beta_t}{\sqrt{1 - \bar{\alpha}_t}}\, \varepsilon\right)$$

让网络预测 $\varepsilon$，则 KL 散度退化为预测均值的 L2 距离，即 MSE。Ho et al. 发现去掉时间步权重系数后（$L_{\text{simple}}$），训练更稳定、效果更好。

### 5.2 其他预测目标对比

除了预测噪声 $\varepsilon$，网络还可以预测：

| 预测目标 | 符号 | 说明 | 适用场景 |
|---------|------|------|---------|
| **噪声（ε-prediction）** | $\hat{\varepsilon} = \varepsilon_\theta(x_t, t)$ | 最常用，DDPM 默认 | 通用 |
| **原图（x₀-prediction）** | $\hat{x}_0 = f_\theta(x_t, t)$ | 直接预测干净图像，两者可互推 | 部分模型、少步采样 |
| **速度（v-prediction）** | $v = \sqrt{\bar{\alpha}_t}\,\varepsilon - \sqrt{1-\bar{\alpha}_t}\,x_0$ | Progressive Distillation 提出，数值更稳定 | 少步蒸馏、高分辨率 |
| **分数（score-prediction）** | $\nabla_{x_t} \log p(x_t)$ | Score Matching 视角，与 ε 差一个尺度因子 | Score SDE 框架 |

三者关系（可互推）：

$$\varepsilon = \frac{x_t - \sqrt{\bar{\alpha}_t}\,\hat{x}_0}{\sqrt{1 - \bar{\alpha}_t}}$$

$$\nabla_{x_t} \log p(x_t) = -\frac{\varepsilon}{\sqrt{1 - \bar{\alpha}_t}}$$

### 5.3 带方差学习的改进损失（Improved DDPM）

DDPM 原始版本固定方差 $\Sigma_\theta = \beta_t \mathbf{I}$。Nichol & Dhariwal 2021 引入**学习方差**：

$$L_{\text{hybrid}} = L_{\text{simple}} + \lambda \cdot L_{\text{vlb}}$$

- $L_{\text{simple}}$：MSE 监督 $\varepsilon$ 预测（主要信号）
- $L_{\text{vlb}}$：完整 ELBO 监督方差学习（stop-gradient 隔离，防止干扰均值学习）

### 5.4 条件生成的损失：Classifier-Free Guidance（CFG）

CFG 在损失层面的做法：训练时以概率 $p_{\text{uncond}}$（通常 10%-20%）随机将条件 $c$（如文本 embedding）替换为空条件 $\varnothing$：

$$L_{\text{cfg}} = \mathbb{E}\left[\left\|\varepsilon - \varepsilon_\theta(x_t, t, c)\right\|^2\right] \qquad \text{（其中 } c \text{ 有 } p_{\text{uncond}} \text{ 的概率被替换为 } \varnothing\text{）}$$

推理时使用线性外插：

$$\hat{\varepsilon}_{\text{guided}} = \varepsilon_\theta(x_t, t, \varnothing) + w \cdot \left(\varepsilon_\theta(x_t, t, c) - \varepsilon_\theta(x_t, t, \varnothing)\right)$$

$w$ 为 guidance scale，越大条件遵从度越强（但多样性降低）。

## 六、优化目标与优化策略

### 6.1 优化目标的本质

从概率角度，整个训练是在做**最大似然估计的变分下界优化**：

$$\max_\theta\; \mathbb{E}_{x_0 \sim p_{\text{data}}}\!\left[\log p_\theta(x_0)\right] \;\geq\; \max_\theta\; \mathrm{ELBO}(\theta) \;=\; \min_\theta\; L_{\text{simple}} \quad\text{（简化后）}$$

更本质的理解（Score Matching 视角，Song et al. 2020）：

$$\min_\theta\; \mathbb{E}_t\!\left[\lambda(t) \cdot \mathbb{E}_{x_0,\, x_t}\!\left[\left\|s_\theta(x_t, t) - \nabla_{x_t} \log q(x_t \mid x_0)\right\|^2\right]\right]$$

即：**学习真实数据分布在每个噪声水平下的分数函数（score function）**，也就是数据分布的梯度场。$s_\theta$ 和 $\varepsilon_\theta$ 差一个符号和尺度：

$$s_\theta(x_t, t) = -\frac{\varepsilon_\theta(x_t, t)}{\sqrt{1 - \bar{\alpha}_t}}$$

### 6.2 优化器与超参数

| 超参数 | 典型值 | 说明 |
|-------|-------|------|
| 优化器 | AdamW | $\beta_1=0.9,\; \beta_2=0.999$ |
| 学习率 | `1e-4` ~ `2e-4` | 配合 warmup |
| LR Schedule | Cosine Decay 或常数 | 通常不用很激进的衰减 |
| Batch Size | 256 ~ 2048 | 越大越稳定，常用梯度累积 |
| EMA | `decay = 0.9999` | 对模型参数做指数移动平均，推理时用 EMA 版本，显著提升质量 |
| 梯度裁剪 | `max_norm = 1.0` | 防止梯度爆炸 |
| 训练步数 | 数十万 ~ 数百万步 | 取决于数据集规模 |

### 6.3 EMA（指数移动平均）的重要性

$$\theta_{\text{ema}} \leftarrow \gamma \cdot \theta_{\text{ema}} + (1 - \gamma) \cdot \theta$$

- 训练过程使用原始参数 $\theta$ 做梯度更新
- 推理 / 评估时使用 $\theta_{\text{ema}}$
- EMA 相当于对历史参数做平滑，显著减少生成图像的高频噪声，是提升 FID 的关键技巧

### 6.4 优化的直观理解

整个训练过程可以理解为：

```
在所有噪声水平 t 上，
教网络学会"如何从一张噪声图中，辨认出混入的噪声方向"

等价于：
在每个噪声水平下，学习真实数据分布的梯度场（指向数据密度增加的方向）

采样时：
从纯噪声出发，沿着梯度场方向反复走小步，逐渐"爬坡"到高密度区域（真实图像）
```

## 七、整体流程总结

```
┌─────────────────────────────────────────────────────────────────┐
│                       训练阶段                                    │
│                                                                   │
│  真实图像 x₀  ──→  随机 t  ──→  闭式加噪得 x_t                   │
│                                    │                             │
│                          U-Net ε_θ(x_t, t)                      │
│                                    │                             │
│                          预测噪声 ε̂                             │
│                                    │                             │
│              Loss = MSE(ε̂, ε)  ──→  AdamW 反向传播更新           │
│                                    │                             │
│                          EMA 更新 θ_ema                          │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│                       推理阶段（DDPM Sampling）                   │
│                                                                   │
│  x_T ~ N(0, I)                                                   │
│      │                                                           │
│  for t = T, T-1, ..., 1:                                        │
│      ε̂ = ε_θ_ema(x_t, t)                                       │
│      x₀̂ = (x_t - √(1-ᾱt)·ε̂) / √ᾱt                           │
│      x_{t-1} = μ_θ(x_t, t) + σ_t · z,   z ~ N(0,I)            │
│      │                                                           │
│  x₀  （生成图像）                                                 │
└─────────────────────────────────────────────────────────────────┘
```


|    经典扩散模型优化的不是高斯噪声参数，而是神经网络本身的参数；高斯噪声只是固定的 “输入信号”，不是可学习参数。

## 八、常见变体速查

| 模型 | 核心改进 | 损失变化 |
|------|---------|---------|
| **DDPM** | 基础范式，ε-prediction | $L_{\text{simple}} = \mathrm{MSE}(\varepsilon,\, \hat{\varepsilon})$ |
| **IDDPM** | 学习方差 + Cosine schedule | $L_{\text{hybrid}} = L_{\text{simple}} + \lambda \cdot L_{\text{vlb}}$ |
| **DDIM** | 确定性采样，不改变训练 | 训练损失不变，采样改为 ODE |
| **LDM / Stable Diffusion** | 在 VAE 隐空间做扩散 | 同 DDPM，但作用于 $z$ 空间 |
| **CFG** | 无分类器引导条件生成 | 训练时随机 drop 条件 |
| **v-prediction** | 预测目标换为速度 $v$ | $L = \mathrm{MSE}(v,\, \hat{v})$ |
| **Flow Matching** | 用 ODE flow 替换扩散过程 | $L = \mathrm{MSE}(u_t,\, v_\theta)$，更简洁 |
